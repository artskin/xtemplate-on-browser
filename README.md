# xtemplate-on-browser

use xtemplate on browser

## prepare environment

```
npm install bower -g
npm install
bower install
cd module
gulp
cd kissy
gulp
```

## use xtemplate with standalone mode

open  ./standalone/demo.html


## use xtemplate with module mode

open ./module/demo.html

## use xtemplate with kissy gallery

open ./kg/demo.html
